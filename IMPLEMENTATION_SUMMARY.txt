🎯 NEW API ENDPOINT
================================================================================

Endpoint: GET /api/analytics/cost-analysis/report

Parameters:
  - start_date (required): Analysis start date (YYYY-MM-DD)
  - end_date (required): Analysis end date (YYYY-MM-DD)
  - format (optional): Report format - 'docx' or 'pdf' (default: docx)
  - departments (optional): Filter by department IDs (comma-separated)
  - price_min (optional): Minimum unit price filter
  - price_max (optional): Maximum unit price filter
  - drug_categories (optional): Filter by drug categories (comma-separated)

Response: Downloadable file (Binary: DOCX or PDF)

Examples:
  - Basic: /api/analytics/cost-analysis/report?start_date=2019-01-01&end_date=2019-12-31
  - PDF:   /api/analytics/cost-analysis/report?start_date=2019-01-01&end_date=2019-12-31&format=pdf
  - Filtered: /api/analytics/cost-analysis/report?start_date=2019-01-01&end_date=2019-12-31&departments=1,2,3&format=pdf
